BLOCK

Dieser Befehl ermöglicht es Ihnen, Ihr Navigations in verschiedene Abschnitte zu organisieren.
Alle Befehle nach diesem Block bis zum nächsten Block werden ein Teil dieses Blocks.

Ein Schaltfläche wird für jeden Block erstellt. Durch anklicken dieser Schaltfläche wird der gluonpilot automatisch zu diesen Block springen.
Durch Drücken von den Anfangsbuchstaben des Blocks (z. B. 's' für 'standby"), wird er das gleiche tun. Drücken Sie die ALT-Taste, um die Shortcuts Ihrer Blocke zu sehen.


Parameters:

  • Name (max 8 zeichen)
