BLOCK

This command allows you to organize your navigation into different sections. All commands after this block (and before the next one), will be part of this block.

A pushbutton will be created for every block statement. Clicking this button will automatically make the gluonpilot jump to this navigation block. Pressing on first letter of the name for this block (eg. 's' for 'Stanby'), will do the same. Press the ALT key to see the shortkeys of your Block-buttons.


Parameters:

  • Name (max 8 characters)
