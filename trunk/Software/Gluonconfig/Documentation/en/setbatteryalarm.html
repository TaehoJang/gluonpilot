SET_BATTERY_ALARM

This command configures the battery voltage alarm. There are 2 kinds of alarms:

  • Warning This will print a warning message every 10 seconds. If you enable speech, the warning message will sound through your speakers.

  • Panic: This will print a panic message similar to the warning message. Optionally the navigation engine can jump to a configurable navigation line at the first occurence of the Panic event.


Example:

This example will circle until the altitude of 100m is reached

  1. FLY_TO(altitude: 200m, ...)

  2. CIRCLE(altitude: 100m)

  3. UNTIL(altitude <= 100m)

  4. FLY_TO(...)


Parameters:

  • Variable

  • Comparator (bigger than, smaller than, …)

  • Value
