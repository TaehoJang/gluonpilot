SERVO TRIGGER

This command allows you to trigger a servo. By triggering we mean:

  • Before the start of the trigger command, the servo is already at a certain position (set by SET SERVO POSITION)

  • The servo trigger command sets the servo at the defined position

  • After a certain delay, the servo's position is set back to the original position


This sequence allows you to, for example, use a servo to push a camera button.


Parameters:

  • Servo channel: 1 to 8. Channels 5 and 6 are usually used. 7 and 8 are available behind number 5 and 6 (and the signal part)

  • Pulse width: a servo is controlled by sending a pulse between 1000 and 2000µs. This value determines the servo's position.
    A value of "0" will output a logical 0 (GND level) on this channel's output. A value of "2500" or higher will output a logical 1 (5V).

  • Position hold: how long is should take before the servo is set back to the original position. The mixing value is 3000 (3 seconds).
