LOITER CIRCLE

This command will fly in a circle around the point that has previously been set with the SET_LOITER_POSITION command.


Example:

This example will circle for 2 minutes around the current position. It should be called with CALL(3)

  1. BLOCK (Loiter)

  2. SET_LOITER_POSITION

  3. LOITER_CIRCLE(100m)

  4. UNTIL(time in block[s] > 120)

  5. RETURN


Parameters:

  • Radius: The circling radius (meter)
